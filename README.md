
# MMPI2中美常模全新界面版本

- 首先带有记忆功能，刷新页面不会丢失数据
- 其次支持JSON和TXT导入/导出
- 然后还有中美常模对比分析
- 响应式界面，支持移动端
- 显示超多量表

## 如何使用
- 在Release里下载`MMPI2双模单文件版.html`然后双击打开
- 所有数据存在浏览器`localStorage`中
